# v-coding-setup

An autonomous AI dev pipeline built on Claude Code — slash-command skills, permission guards and hooks, scheduled rituals (harvesters, daily plan/summary, periodic review), a Linear-backed ticket flow, and a markdown-in-git knowledge base.

> **Status: work in progress.** This repo is being extracted from a private, battle-tested setup (~760 commits of self-development). Not yet installable. See [docs/plan.md](docs/plan.md) for the build plan.

Companion repo: [linear-mcp-lean](https://github.com/wiklob/linear-mcp-lean) — the self-hosted, token-frugal Linear MCP server this pipeline pairs with.
