# Parked — ideas snoozed, not killed

Ideas deliberately set aside: revisitable, not declined. The **set** is anchored to Linear's parked state (`/align` parks an Intake idea by moving it to `Backlog`); the **why-parked** note here is authored — that reasoning doesn't live in the tracker. Killed ideas are *not* here (they stay Canceled in Linear with a trace comment).

When reviving a parked idea, move it forward in Linear and add a dated line to [`decisions.md`](./decisions.md).
